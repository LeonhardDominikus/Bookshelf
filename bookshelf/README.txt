#Bookshelf Database

## Introduction

## Setup
Before running the project, make sure you have the necessary files in the correct directories:

1. Images : All image files should be placed in the `img` directory. The images are used for various features in the project. Accepted formats include `.jpg` and `.png`.

2. PDFs : All PDF files should be placed in the `pdf` directory. These files are used for the download feature in the project.

Please ensure that the `img` and `pdf` directories exist at the root level of the project directory. If they do not exist, you will need to create them.

#Database Bookshelf

## Pendahuluan
Proyek ini adalah aplikasi web yang berfungsi sebagai sistem manajemen buku. Aplikasi ini memiliki fitur login yang aman dan database buku yang memungkinkan operasi CRUD (Create, Read, Update, Delete) dasar.

## Persiapan
Sebelum menjalankan proyek, pastikan Anda memiliki file yang diperlukan dalam direktori yang benar:

1. Gambar : Semua file gambar harus ditempatkan dalam direktori `img`. Gambar digunakan untuk berbagai fitur dalam proyek. Format yang diterima termasuk `.jpg` dan `.png`.

2. **PDF**: Semua file PDF harus ditempatkan dalam direktori `pdf`. File ini digunakan untuk fitur unduhan dalam proyek.

Pastikan direktori `img` dan `pdf` ada di level root dari direktori proyek. Jika mereka tidak ada, Anda perlu membuatnya.
